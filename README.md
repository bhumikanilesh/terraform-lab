# Terraform Lab - Infrastructure as Code

## Objective
This lab demonstrates Infrastructure as Code (IaC) using Terraform. It includes creating, managing, and automating infrastructure using configuration files.

## Tools Used
- Terraform CLI
- Docker
- Git & GitHub

## Terraform Workflow
- terraform init
- terraform plan
- terraform apply
- terraform destroy

## Project Structure
terraform-lab/
├── day1
├── day2
├── day3
├── day4
├── day5

## Assignments

### Day 1
Basic Terraform configuration using local provider to create a file.

### Day 2
Using variables and tfvars files for different environments.

### Day 3
Managing Docker container using Terraform.

### Day 4
Using modules and dynamic infrastructure creation.

### Day 5
CI/CD pipeline using GitHub Actions.

## Bonus
Dynamic creation of multiple containers using for_each.

## Conclusion
This lab demonstrates Terraform fundamentals, modular design, container orchestration, and CI automation.
